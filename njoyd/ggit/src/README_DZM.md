# ggit

## 一、使用

- 安装 (`小提示：记得切换为私有源再进行安装`)

  ```sh
  # 安装并切换好了可忽略这一步
  # 安装 nrm 管理镜像源
  $ npm install -g nrm
  # 添加私有镜像源
  $ nrm add nnpm http://10.0.10.23:7001
  # 使用私有镜像源
  $ nrm use nnpm
  ```

  ```sh
  $ npm i @nj/base
  ```

- 引入

  ```javascript
  // 方式一
  import * as Base from "@nj/base";

  // 方式二
  import { xxx, xxx, xxx } from "@nj/base";
  ```
